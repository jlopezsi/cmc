## Conventions

  * Files and directories at the root folder have the .prjid extension (e.g.
    `library.gnr.R` where `library.R` is the original name and `gnr` is the is
    project id. 
  * There is a `config.R` soft link pointing to the main config file.

## Directories

  * `archive`     : anything not current
  * `data`        : raw or processed data
  * `library`     : any library used in the current project
  * `literature`  : related papers
  * `manuscript`  : manuscript development file containing latex and Rmd files
  * `playground`  : laboratory of r codes
  * `proceeding`  : abstracts, presentations and other files generated for the conferences
  * `publication` : submission and revision files
  * `report`      : temporary report files generated by r
  * `r`           : the project engine


### Prefixes

  * `arr.`     : arrays
  * `cnt.`     : counter
  * `dir.`     : directory addresses
  * `df.`      : data frames
  * `fil.`     : files or file addresses
  * `fun.`     : function name
  * `prj.`     : project config variables
  * `session.` : session config variables
  * `str.`     : strings
  * `vct.`     : vector
